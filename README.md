# Week 1 React Assignment - Simple Reusable Component

 Project Goal
This project demonstrates creating a reusable React component and using it multiple times with different props.

 Component
- Component Name: Badge
- Props Used:
- label (text)
- value (text or number)

 Usage
The Badge component is rendered at least 4 times in App.jsx with different props:

jsx
<Badge label="Course" value="React" />
<Badge label="Week" value="1" />
<Badge label="Topic" value="Props" />
<Badge label="Difficulty" value="Easy" />
